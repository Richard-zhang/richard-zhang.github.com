Hack C++ by myself(0)

Before I start, I would like to share some feeling about C++ practice. I used to be enthusiastic in finding “best” C++ book, trying to learn all C++ language by reading. Obviously, it’s useless. Time is consuming and after reading you found out you forget everything.

I prefer study method which is problem-motivated. When I try to solve a specific problem, I will try to think how to do effectively. With the help of search engine dash and books(I prefer to use books as reference instead of learning material), I gradually have a better command of the language by solving problems one by one.

To start my problem-motivated method, I choose hackerank and 计蒜客 as question banks.

This is the first article of Hack C++ by myself series. I hope I can learn one new trick per day as well improving my ability in expanding.
